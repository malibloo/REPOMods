# REPOMods
Some cobbled together mods for the game R.E.P.O.

## Mali's Map Modules
Multiple modules that give the map more functionality
Some intended to also optionally implement them as upgrades
- Valuables spy - Client
  Show all valuables from the start. Toggleable.
- Enemies spy - Client
  Show enemies on the map. Toggleable, color configurable.
  (Experimental) Use Enemy Rigidbody option to have more accurate locations on the map. Might not work well on clients.

## Mali's Tweaks
Small tweaks meant to customize your playstyle. Mostly meant to make the game slightly fairer.
- Equippable Zero Grav - Host enforces
  Makes it so that items being pulled out of the inventory into the world get a moment of zero gravity, so they don't accidentally fall, make a noise, or fire by themselves when you have a laggy player with you.
- Enemy Valuable - Host enforces
  Enemy valuables (souls, crystals) have 5 seconds of invulnerability. With this mod, if it falls in a pit during this time, it'll reset the timer.
