# REPOMods
Some cobbled together mods for the game R.E.P.O.

## Mali's Map Modules
Multiple modules that give the map more functionality
Some intended to also optionally implement them as upgrades
- Valuables spy (Client) - Show all valuables from the start. Key toggleable.
- Enemies spy (Client) - Show enemies on the map. Key toggleable, color configurable. (Experimental) Use Enemy Rigidbody option to have more accurate locations on the map. Might not work well on clients.

## Mali's Tweaks
Small tweaks meant to customize your playstyle. Mostly meant to make the game slightly fairer.
- Equippable Zero Grav (Host enforces) - Makes it so that items being pulled out of the inventory into the world get a moment of zero gravity. This will prevent the item falling, making a noise, or firing by themselves when you have a laggy player with you. Config toggleable, timer configurable (default is 0.5).
- Enemy Valuable Pit Invulnerability (Host enforces) - Enemy valuables (souls, crystals) have 5 seconds of invulnerability. With this mod, if it falls in a pit during this time, it'll reset the timer. No more losing them due to bad pit RNG. Config toggleable, timer configurable (default is 5.0).
